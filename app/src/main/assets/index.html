SAFE_UP

Safety first is safety always.

  • Introduction

    SAFE-UP is a safety app. By using this app one can easily send a message and location to their selected contacts in case of any emergency by just clicking on "HELP" button on Welcome Screen.

  • HOW TO USE

    Configure the app(For first time users)
    1. Install the application.
    2. Continue by accepting all the permissions.(If u don't accept the permissions the app will not work furthur.)
    3. Press "ADD CONTACTS" button (on the home screen) and add the emergency contacts from your storage on which you want to send the emergency message.
    4. Press "ADD TEMPLATES" button (on the home screen) and add the templates you want to send in case of any emergency.
    5. Set any one template as default template.The selected template will be delivered automatically in case of emergency.
    6. All the above steps are to be followed when the user is using the app for the first time.
    After the app is configured:
    1. Mobile's location(GPS) should be turned on.
    2. In case of any emergency, the user just have to open the app.
    3. Press "HELP" button.
    4. After pressing "HELP" button, the template that the user selected as default, will be delivered to all the contacts which were added while configuring the app.
    5. With message, the location will also be delivered.
    6. The message will contain a link. By opening the link, that particular location will be shown on the Google Maps.
